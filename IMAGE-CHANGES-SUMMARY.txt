========================================
HOW TO USE
========================================

OPTION 1 - Replace Files Directly:
1. Go to ISONMICHS/img/ folder
2. Replace any image with your own
3. Keep the same filename
4. Refresh browser (Ctrl+F5)

OPTION 2 - Change Filenames in Code:
1. Add your image to ISONMICHS/img/ folder
2. Open solar-enhancements.css
3. Find the comment "TO CHANGE:"
4. Replace the filename
5. Save and refresh browser
