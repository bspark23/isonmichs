========================================
HOW TO REPLACE AN IMAGE
========================================

METHOD 1 - Replace the file directly:
1. Find the image you want to replace in the "img" folder
2. Delete or rename the old image
3. Copy your new image into the "img" folder
4. Rename your new image to match the old filename exactly
5. Refresh your browser (Ctrl+F5 to clear cache)

METHOD 2 - Change the filename in the code:
1. Copy your new image into the "img" folder
2. Open the CSS file (solar-enhancements.css) or HTML file (index.html)
3. Find the image reference (look for the comments "TO CHANGE:")
4. Replace the filename with your new image filename
5. Save the file and refresh your browser

========================================
IMAGE REQUIREMENTS
========================================

Hero Carousel Images:
  - Recommended size: 1920x1080 pixels (landscape)
  - Format: JPG or JPEG
  - File size: Under 500KB for fast loading

Page Header Images:
  - Recommended size: 1920x600 pixels (wide landscape)
  - Format: JPG or JPEG
  - File size: Under 300KB

Service Images:
  - Recommended size: 800x600 pixels
  - Format: JPG or JPEG
  - File size: Under 200KB

========================================
TIPS
========================================

1. Always keep a backup of your original images
2. Use descriptive filenames (e.g., "solar-installation-2024.jpg")
3. Optimize images before uploading to reduce file size
4. Test on mobile devices after changing images
5. Clear browser cache (Ctrl+F5) if images don't update

========================================
NEED HELP?
========================================

All image paths in the CSS file are marked with comments:
  /* TO CHANGE: Replace 'filename.jpg' with your own image */

Look for these comments to find exactly where to change images.
